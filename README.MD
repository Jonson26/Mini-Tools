# Mini Programs

This repository contains a bunch of small bits and bobs I wrote for my own use. They come with absolutely no warranty at all!

## RPN-CLI

A Reverse Polish Notation calculator that runs in the command line and has a nice TUI. Requires Python 3.

## Pingen

A simple python script that can generate a random PIN. Requires Python 3.

## MonoIMG

A small GUI app that allows you to convert images to monochrome resized versions suitable for really old PC's while making use of Pillow's excellent dithering algorithm. Requires python3, tkinter and pillow.

## German US Keyboard

A keyboard layout I made a long time ago using Microsoft's ancient layout editor thingy. It is a QWERTY based on the Polish Programmer's layout that allows one to write German diacritic characters in much the same manner. For example, alt+a (or pressing '~' and then 'a') writes an ä character. The Caps Lock and Shift keys are also fully supported.

This is only the installer package. If I ever find the source files, I'll put them here as well.
